# Instructions

After downloading files, run within the directory:
>npm install

To start the server, run the following code within an opened terminal:
>npm run dev

Once the server has started, it will output a local address like the one below:
>http://localhost:5173/

## Game Play

When the address is opened, a screen with 3 sections will appear:

The display section, which shows both the player's choice and the enemy's choice

The choice section, which includes buttons to choose your attack then buttons to play and reset the board

The last section is the results seciton. This section displays the results as well as any streak a player might obtain
